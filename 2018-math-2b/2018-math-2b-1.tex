    \section{第1問}
    \subsection{問題について}
    [1]は三角関数の加法定理、合成の理解を問う問題。しかし、冒頭にラジアンの定義に関する設問があり、人によっては出鼻をくじかれる形になったかもしれない。(3)で変数の変換が行われているので逐一変数の定義域を確認して解き進めよう。

    [2]は対数の理解を問う問題。$\log_{3}{x}$は$x>0$の時、実数全体をとることから$t$に置き換え、$t$についての方程式を解くように誘導されている。このような方法は対数を扱う常套手段であるため身につけてもらいたい。

    \subsection{解答例}
    \begin{description}
        \item[ア] 答えは２である。\\
        ラジアンの定義で大事な発想は、半径が同じ扇型においては中心角の大きさの比と弧の長さの比は同じであるということである。そこでもう一度答えの定義を考えると、扇型(半径１)の中心角の大きさ(ラジアン)と弧の長さの値が一致していることが容易にわかり、これがとても便利なのだ。
        \item[イ,ウ] 直前の問題より半径１で中心角が$144^{\circ}$の扇型の弧の長さを求めればいい。
        \[
        2\pi \times \frac{144}{360} = \frac{4}{5} \pi
        \]
        \item[エ-カ] 先の問題の逆の計算をすればよいが少し楽がしたい。$\pi \hspace{1mm}(ラジアン) =180 ^{\circ}$だから
        \begin{eqnarray*}
            \frac{23}{12} \pi \hspace{1mm} (ラジアン) &=& \frac{23}{12} \times 180^{\circ}\\
            &=&345^{\circ}
        \end{eqnarray*}
        \item[キ]
        \begin{eqnarray*}
            \theta +\frac{\pi}{30} &=& \left( \theta +\frac{\pi}{5} \right) - \frac{\pi}{6}\\
            &=& x- \frac{\pi}{6}
        \end{eqnarray*}
        \item[ク] まず$\cos$だけ加法定理を適応する。
        \begin{eqnarray*}
            \cos \left( x- \frac{\pi}{6} \right) &=& \cos x \cos \left( - \frac{\pi}{6} \right) -\sin x \sin \left( - \frac{\pi}{6} \right)\\
            &=& \frac{\sqrt{3}}{2}\cos x +\frac{1}{2}\sin x
        \end{eqnarray*}
        あとは元の式に代入
        \[
        2\sin x-2 \cos \left( x- \frac{\pi}{6} \right) = \sin x -\sqrt{3} \cos x
        \]
